# 🔮 2026 예언기

Next.js(App Router) + TypeScript + Tailwind CSS로 만든 모바일 우선 웹앱입니다.

## 🚀 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 📱 기능

- 이름 입력 후 랜덤 예언 생성
- 모바일 우선 반응형 디자인
- 눈 내리는 애니메이션 & 별 반짝임 효과
- 결과 공유 기능 (Web Share API 또는 클립보드 복사)

## 🛠 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## 📦 배포

Vercel에 배포하려면:

```bash
npm run build
```

그리고 Vercel에 프로젝트를 연결하세요.
